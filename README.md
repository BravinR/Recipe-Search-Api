## Recipe Search API
The Website allows users to search for recipes from Thousands of dishes using the Edemam API.

The project uses **REACT JS, Hooks(state/effect), Asynchronous API(asyn/await,JSX)**

Custom CSS is used to style the website and to make it easy to manage CSS, **CSS modules** are used

The website is deployed to Netlify [Check it Out](https://recipepro.netlify.app)
